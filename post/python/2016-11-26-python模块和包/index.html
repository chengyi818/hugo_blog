前言

最近在写一个Python的工具,为了使代码清晰易阅读,我分了若干个模块.模块间是有层次关系的,所以此时有必要引入"包(package)".

通过一个小的Demo展示了如何在包和模块间互相引用

我们囊括如下几种情况:

  • 引用子目录的模块
  • 引用同级模块
  • 引用不同目录的模块

包(Package)

包是若干模块的一个集合,一个文件夹变成包的方法就是在其中放入__init__.py文件.在导入包的时候,该文件将会首先执行文件.

所以,在我们的代码中,subpackage1和subpackage2是两个包.
